ZeroExif — это мощный кроссплатформенный скрипт, созданный для очистки изображений от EXIF, XMP, ICC и других скрытых данных. Он защищает вашу приватность, удаляя GPS-координаты, модели камер и точное время съемки перед тем, как вы опубликуете фото в сети.

Скрипт оснащен собственным движком отрисовки интерфейса прямо в терминале, поддерживает Drag & Drop (перетаскивание файлов) и имеет встроенную мультиязычность.

---

## 📥 Какую версию скачать?

В репозитории находятся две версии скрипта. Это сделано для обеспечения идеальной работы на абсолютно любых устройствах. Выберите ту, которая подходит вам:

| Файл | Для чего подходит | Описание |
| :--- | :--- | :--- |
| **`ZeroExif.py`** | **Windows, macOS, Linux** | Содержит интерактивное всплывающее меню настроек. Идеально для десктопных операционных систем. |
| **`ZeroExif_NO_TUI.py`** | **Termux (Android)** | Облегченная версия без сложных всплывающих элементов. <br>⚠️ **Важно для Termux:** Интерактивное меню из основной версии не работает в среде Termux, из-за чего вы не сможете зайти в настройки. Обязательно используйте эту версию на смартфонах! |

## 🚀 Установка

Убедитесь, что на вашем устройстве установлен Python.

**1. Клонируйте репозиторий:**
```bash
git clone https://github.com/Datvex/ZeroExif.git
cd ZeroExif
```

**2. Установите зависимости:**
Для работы с изображениями скрипту требуется библиотека `Pillow`.
```bash
pip install -r Requirements.txt
```

**3. Запустите скрипт:**
```bash
python ZeroExif.py
```

**3.1 Запустите скрипт (Для Termux):**
```bash
python ZeroExif_NO_TUI.py
```

## 🕹️ Управление и Навигация

Управление в ZeroExif полностью оптимизировано под клавиатуру.

### Главное меню
Используйте цифры на клавиатуре для выбора пунктов.
* <kbd>1</kbd> **Начать очистку:** Переход к выбору фотографий.
* <kbd>2</kbd> **Настройки:** Изменение пути сохранения очищенных фото и смена языка (English, Русский, 中文).
* <kbd>Esc</kbd> **Выход:** Безопасное закрытие программы.

### Этап выбора файлов
Вам не нужно вписывать пути к файлам вручную. Просто **перетащите папку или файлы (Drag & Drop)** прямо в окно терминала.

| Действие | Клавиша / Ввод |
| :--- | :--- |
| **Выбрать/Отменить файл** | <kbd>1</kbd> – <kbd>10</kbd> (Цифра напротив файла) |
| **Следующая страница** | <kbd>N</kbd> (Next) |
| **Предыдущая страница** | <kbd>P</kbd> (Previous) |
| **Добавить еще фото** | Перетащите новые файлы в терминал |
| **Продолжить (к очистке)** | <kbd>0</kbd> |
| **Вернуться назад** | <kbd>Esc</kbd> |

### Выбор удаляемых данных
Вы можете настроить, какую именно информацию нужно стереть. Переключайте опции соответствующими цифрами:

1. **Все метаданные** (Рекомендуется) - Удаляет EXIF, XMP, ICC.
2. **GPS / Координаты** - Стирает только местоположение.
3. **Информация о камере** - Удаляет модель устройства и параметры объектива.
4. **Дата и время** - Удаляет временные метки создания фото.

Нажмите <kbd>0</kbd>, чтобы запустить процесс удаления.

## 🛠️ Решение проблем (Частые вопросы)

> **Ошибка: Библиотека Pillow не установлена**
> Выполните команду `pip install Pillow`. Если вы используете Linux, возможно потребуется команда `pip3 install Pillow`.

> **В Termux не открываются настройки / Зависает интерфейс**
> Вы запустили файл `ZeroExif.py`, который содержит специфичный для ПК графический интерфейс. Пожалуйста, используйте файл `ZeroExif_NO_TUI.py`, он создан специально для корректной работы в Termux.

> **Ошибка: Нет доступа к папке (PermissionError)**
> Убедитесь, что у скрипта есть права на запись в папку, указанную в "Пути сохранения". В Termux необходимо предварительно выдать доступ к хранилищу командой `termux-setup-storage`.
